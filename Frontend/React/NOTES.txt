-- When we are returning the elements which are wrapped inside another element. They both are child to the wrapped element, but if you want to have both the child elements act as individual elements without the parent element then we have to use jsx fragment.

jsx frament -- <></> -- dummy tag which will be ignored by the browser when rendering the elements so that when we are returning the elements we are going to wrap it inside the jsx fargment hence we are returning only one element as per jsx but when rendering the elements the browser will ignore the fargment and only considers the child elements.
